# Anthropic High Contrast Theme

A beautiful, high-contrast theme for Visual Studio Code inspired by [Anthropic's](https://anthropic.com/) distinctive brand colors. It features two variants (Dark and Light) built for optimal readability with Anthropic's signature aesthetics.

## Features

- **Dark & Light Variants**: Includes both "Anthropic High Contrast" (Dark) and "Anthropic High Contrast Light" to seamlessly match your system's appearance.
- **High Contrast**: Designed meticulously for legibility to reduce eye strain, adhering to strong WCAG contrast ratios.
- **Warm Dark Palette**: Instead of a stark black or cold blue-grey, it uses warm off-black and charcoal tones, emulating a modern, comfortable development environment.
- **Coral Accent**: Highlights like search matches, focus outlines, and important language keywords utilize Anthropic's distinct coral `#DA7756` color.
- **Balanced Syntax Highlighting**: Soothing, discernible syntax colors that pop against the background without being overly completely saturated.

## Installation
*(Since this is a local development version)*

1. Navigate to the extension folder: `d:\work_space\vscode-anthropic-theme`
2. Open it in VS Code: `code .`
3. Press `F5` to open the Extension Development Host.
4. Go to Preferences: Color Theme (`Ctrl+K Ctrl+T`) and select **Anthropic High Contrast**.

Enjoy coding in a comfortable, Anthropic-inspired environment!
